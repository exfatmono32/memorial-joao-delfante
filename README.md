# Memorial João Delfante

Projeto de site estático pronto para publicar no GitHub Pages.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie todos os arquivos desta pasta para o repositório.
3. No GitHub, abra **Settings**.
4. Vá em **Pages**.
5. Em **Build and deployment**, escolha:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/root`
6. Salve e aguarde o link do site ser gerado.

## Arquivos principais

- `index.html` → página inicial
- `styles.css` → estilo visual
- demais `.html` → páginas das seções

## Personalização

Você pode editar qualquer texto direto nos arquivos `.html`.
